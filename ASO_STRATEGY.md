## Landing Page SEO Strategy

- **H1 Optimization:** Must contain the primary, highest-value keyword cluster (e.g., "PostCraft AI: AI Content Generator for Social Media")
- **Service Page Depth:** Create dedicated sub-pages for specific use cases (e.g., `/instagram-captions`, `/blog-ideas`, `/linkedin-posts`). This allows targeting long-tail keywords for each platform
- **Content Cluster Strategy:** Develop a comprehensive "Resource Center" blog section. Pillar content (e.g., "The Ultimate Guide to Social Media Content Marketing in 2024") linked to multiple cluster articles (e.g., "Best Hooks for Instagram Reels," "SEO Tips for Blog Writers")
- **Schema Markup:** Implement **Product Schema** (for the tool itself) and **FAQ Schema** (answering common questions like, "Is PostCraft AI good for small businesses?")
- **E-E-A-T Signals:** Prominently display authority signals (e.g., "Trusted by marketers at [Industry X]," "Used by 10,000+ creators") to build trust and search engine confidence
